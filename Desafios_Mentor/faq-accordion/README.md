# Frontend Mentor - FAQ accordion

Proyecto : Acordeón de preguntas frecuentes.

Realizado con HTML, CSS Y JavaScript.

Para la navegación con el ratón hay que posicionar el cursor sobre la pregunta, y activarla con un click.
 
Para la navegación con el teclado hay que utilizar las teclas de las flechas. Luego de seleccionar
la pregunta, se activa con la tecla Enter.

Para el diseño responsivo  se consideraron cuatro tamaños de ventana, desde el tamaño de pantalla de 
un dispositivo movil (menores a 500 pixeles), hasta pantallas de más de 1200 pixeles.
